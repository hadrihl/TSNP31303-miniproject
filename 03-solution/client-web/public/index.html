About

This system provides sensors monitoring from Edison SN (Sensor Node) from outdoor environment. Sensor data such as Ultra-voilet light sensor, humidity sensor, and temperature sensor are gathered together and store into ultra-lightweight Redis server and MySQL server on the web. A simple analyics is performed by the web providing realistic analysis of sensors information.
